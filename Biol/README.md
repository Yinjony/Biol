# 生物做题小程序

这是根据 `生物做题小程序.md` 实现的 V0.1 Web 版 MVP，可直接在浏览器打开 `index.html` 使用。

## 已实现

- 题库新增、修改、删除、模糊查询
- 选择题和判断题两种题型
- OCR/Word 文本粘贴或文本文件读取后的智能填入
- 随机抽取 10/25/50 道题
- 作答后立即判断对错，并锁定答案
- 提交时检查未作答题号，完成后统计做对/做错数量
- 本地 `localStorage` 存储题库数据

## 文件

- `index.html`：页面结构
- `styles.css`：界面样式
- `app.js`：题库和练习逻辑
- `schema.sql`：PostgreSQL 建表脚本

## 后续接入方向

当前版本用浏览器本地存储模拟数据库。后续如果要按需求文档升级为 Uni-app + Vue3 + Pinia + Node.js/TS + PostgreSQL，可以复用现有交互流程，把 `app.js` 中的本地增删改查替换为后端接口。
